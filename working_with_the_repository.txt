1. Переходим в рабочую папку через команду - cd
2. Создаем директорию                      - mkdir <имя>
3. Переходим в созданую директрию          - cd <имя>
4. Инициализируем                          - git init
5. Определяем текущее состояние            - git status
6. на GitHub уже зарегистрированы
